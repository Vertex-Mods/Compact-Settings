## âš™ï¸ Installation

### Sine

1. Install latest version of [Sine](https://github.com/CosmoCreeper/Sine) (if you haven't already).
2. Restart Zen Browser.
3. Open settings and go to the `Sine` tab.
4. Search for `Compact Settings`.
5. Click Install.
6. Check pages `about:preference` and `about:addons` to see changes.

## âš™ï¸ Configuration

You can customize the script's behavior via `about:config` or the settings UI in Sine.

- `compact-settings.enabled`
- `compact-settings.animation.enabled`
- `compact-settings.compact-addons.enabled`
- `compact-settings.addons.grid-enabled`
